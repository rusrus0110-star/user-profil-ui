# Profile Card Playground

Interactive React + MUI application for customizing and previewing a user profile card in real time.

## ✨ Features

- Live editing of user profile (name, surname, role)
- Adjustable avatar size
- Online / offline status toggle
- Button customization (color and size)
- Card style switch (elevation / outlined)
- Responsive 40/60 layout
- Clean and aligned UI based on design reference

## 🛠 Tech Stack

- React (functional components + hooks)
- Material UI (MUI v5)
- Flexbox layout

## 🚀 Getting Started

```bash
npm install
npm run dev
```
